# 阶段 11 功能验证指南

> **完成时间**: 2026-05-29  
> **功能模块**: 中译英写作增强与反向检索联想

---

## 📋 功能清单

### ✅ 功能 1: AI 教师中文同义词辨析模式
**实现内容**: 当在侧边栏查词框输入中文词汇时，AI 教师会自动切换到"同义词精细语境辨析"模式，提供多个英文表达的对比分析。

### ✅ 功能 2: 编辑器内中译英写作联想补全
**实现内容**: 在编辑器中输入 `::` 前缀 + 中文词汇，会触发智能联想，推荐对应的英文单词（优先推荐生词本中的词汇）。

---

## 🧪 验证步骤

### 一、AI 教师中文同义词辨析验证

#### 步骤 1: 打开插件侧边栏
1. 在 Obsidian 中打开任意笔记
2. 点击右侧边栏的 **Language Learner** 图标（或使用快捷键）
3. 确保侧边栏面板已展开

#### 步骤 2: 配置 AI 教师（如果尚未配置）
1. 在侧边栏顶部找到 **AI 教师配置** 折叠面板
2. 点击展开，填入以下信息：
   - **API Key**: 您的 DeepSeek API Key（或其他兼容 OpenAI 格式的 API Key）
   - **Base URL**: `https://api.deepseek.com/v1`（或您的 API 服务地址）
   - **Model**: `deepseek-chat`（或您选择的模型）
3. 配置完成后折叠面板

#### 步骤 3: 测试中文查询
1. 在侧边栏顶部的 **全局查词输入框** 中输入中文词汇，例如：
   ```
   掌握
   ```
2. 点击 **🔍 查询** 按钮
3. 系统会显示多个英文候选词（如 `master`, `grasp`, `command` 等）
4. 点击任意一个候选词，进入单词详情页

#### 步骤 4: 触发 AI 教师辨析
1. 在单词详情卡片中，找到 **🤖 AI 教师** 按钮
2. 点击该按钮
3. **预期结果**:
   - AI 教师会返回针对"掌握"这个中文词汇的详细辨析
   - 包含 3-5 个常用英文表达（如 master, grasp, command, control 等）
   - 每个表达都有：
     - 适用场景（正式/非正式、口语/书面）
     - 语义侧重点
     - 常见搭配
     - 地道造句示例
   - 最后给出选词建议

#### 步骤 5: 对比英文查询模式
1. 清空查词框，输入英文单词，例如：
   ```
   master
   ```
2. 点击查询，进入单词详情
3. 再次点击 **🤖 AI 教师** 按钮
4. **预期结果**:
   - AI 教师会返回传统的词根词源解析
   - 包含词根拆解（如 master 的词源）
   - 常用词组/固定搭配
   - 精巧造句示例

### 二、编辑器内中译英写作联想验证

#### 步骤 1: 打开编辑器
1. 在 Obsidian 中新建或打开一个 Markdown 笔记
2. 确保光标在编辑区域内

#### 步骤 2: 测试中译英联想（精确匹配）
1. 在编辑器中输入：
   ```
   ::掌握
   ```
2. **预期结果**:
   - 输入 `::掌` 时，联想框会自动弹出
   - 显示与"掌握"相关的英文单词列表
   - 每个单词显示：
     - 单词本身 + 音标
     - **🌐 中译英** 标签（蓝色）
     - 中文释义
     - 状态标签（📌 生词 / ⚡ 学习中 / ✅ 已掌握）
   - 优先级排序：学习中 > 生词 > 已掌握 > 其他

3. 使用 **↑↓ 方向键** 选择候选词，按 **Enter** 或点击鼠标
4. **预期结果**:
   - 选中的英文单词会替换 `::掌握`
   - 单词后自动添加一个空格
   - 光标移动到单词后，方便继续输入

#### 步骤 3: 测试中译英联想（模糊匹配）
1. 在编辑器中输入：
   ```
   ::重要
   ```
2. **预期结果**:
   - 显示与"重要"相关的英文单词（如 `important`, `crucial`, `vital`, `significant` 等）
   - 同样带有 **🌐 中译英** 标签

#### 步骤 4: 测试英文联想（对比）
1. 在编辑器中直接输入英文前缀（不带 `::`）：
   ```
   mas
   ```
2. **预期结果**:
   - 显示以 `mas` 开头的英文单词（如 `master`, `mass`, `massive` 等）
   - 标签为：🔥 临界复习 / 📌 生词 / ⚡ 学习中 / ✅ 已掌握
   - **没有** 🌐 中译英 标签

#### 步骤 5: 测试优先级排序
1. 先在侧边栏查词框中搜索 `::掌握`，将某个候选词（如 `master`）标记为 **学习中**
2. 再次在编辑器中输入 `::掌握`
3. **预期结果**:
   - `master` 会排在候选列表的最前面（因为状态为 LEARNING）
   - 带有 **⚡ 学习中** 标签

## 🐛 常见问题排查

### 问题 1: AI 教师返回的仍然是词根解析，而非同义词辨析
**原因**: 中文检测逻辑未生效  
**排查**:
1. 检查查询的词汇是否包含中文字符
2. 查看浏览器控制台（Ctrl+Shift+I），确认 `isChinese` 参数是否为 `true`

### 问题 2: 输入 `::掌握` 后没有弹出联想框
**原因**: 反向索引未加载或前缀匹配失败  
**排查**:
1. 确认 `REVERSE_INDEX` 中是否包含"掌握"这个词条
2. 检查是否输入了完整的 `::`（两个冒号）
3. 查看控制台是否有报错信息

### 问题 3: 联想框显示的单词没有中文释义
**原因**: 离线词典未覆盖该单词  
**解决**: 这是正常现象，系统会在后续查询时自动补充释义
