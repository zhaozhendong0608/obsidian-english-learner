# Obsidian English Immersion Reader (英语沉浸式精读助手)

Obsidian English Immersion Reader 是一款专为英语学习者打造的**内存优先、高性能、沉浸式**的本地化英语精读与视听学习插件。它能将您的 Obsidian 库变成一个高度互动的“外刊精读室”和“视听影音库”。

---

## 🎨 核心感官特性

*   **轻松冷启动**：通过约 20 道词汇测试题（自适应二分查找算法），两分钟迅速估算您的英语词汇量，并一键自动隐藏所有背景熟词，实现视觉降噪。
*   **沉浸式精读**：鼠标悬停在单词或**多词短语**（如 *look forward to*）上，零延迟秒现音标和释义；已掌握熟词在 GPU 加速下渐进半透明褪色，让您的注意力聚焦在生词上。
*   **交互式视听笔记 (Media Extended)**：支持本地视频、YouTube 和 B 站内嵌页。在记笔记时可一键插入带时间戳的超链接，双向点击即可精确定位跳转播放；大字字幕看板支持单词直接点击发音查词。
*   **影子陪伴真人朗读**：集成有道/谷歌双引擎发音及本地离线 TTS 兜底。长句朗读时在停顿处自然意群换气，并能根据单词难度自适应调节语速。
*   **科学卡片复习与写作补全**：自动将查词所在的完整句子提取为独立的生词卡片（追加并不破坏手写内容），并以经典的 **SM-2 间隔记忆算法**调度复习。在 Obsidian 撰写英文日记时，输入法会自动优先联想补全您的生词。

## 🛠️ 本地安装与开发构建指南

如果您想在本地构建该插件并安装到 Obsidian 中，请按照以下步骤操作：

### 第一步：准备开发环境
请确保您的电脑上已经安装了 [Node.js](https://nodejs.org/)（建议版本 v18 或以上）。

### 第二步：下载插件源码并安装依赖
1. 克隆或下载本仓库源码到本地：
   ```bash
   git clone https://github.com/zhaozhendong0608/obsidian-english-learner.git
   cd obsidian-english-learner
   ```
2. 安装项目依赖：
   ```bash
   npm install
   ```

### 第三步：编译打包插件
在项目根目录下运行打包命令：
```bash
npm run build
```
编译成功后，项目根目录下会生成/合并出以下三个核心文件（即 Obsidian 插件的运行资产）：
*   `manifest.json`
*   `main.js`
*   `styles.css`

## 🔌 在 Obsidian 中手动启用插件

1.  **找到插件安装路径**：打开您需要使用该插件的 Obsidian 库（Vault），在您的电脑文件管理器中定位到 `.obsidian/plugins/` 目录。
    *(注：以 `.` 开头的文件夹在系统中默认是隐藏的，Windows 用户请勾选“显示隐藏项目”，macOS 用户请使用快捷键 `Cmd + Shift + .` 显示隐藏文件。)*
2.  **创建插件文件夹**：在 `plugins/` 目录下，新建一个命名为 **`obsidian-english-immersion-reader`** 的文件夹。
3.  **放入资产**：将您在[第三步]中生成的 `manifest.json`、`main.js` 以及 `styles.css` 这三个文件拷贝至新建的 `obsidian-english-immersion-reader` 文件夹中。
4.  **在 Obsidian 中启用**：
    *   打开 Obsidian 的 **“设置” (Settings)** ➡️ **“第三方插件” (Community plugins)**；
    *   点击“重新加载” (Installed plugins 旁的刷新图标)；
    *   在已安装列表中找到 **Obsidian English Immersion Reader**，将其开关打开启用。

## 🚀 极速上手使用

### 1. 估算您的初始词汇量 (评估)
启用插件后，在右侧侧边栏中找到插件控制台，切换至 **“估算”** 选项卡。跟随提示测试 20 个单词。测试完毕后，点击一键确认，水位线以下的常见词汇将被系统自动标记为“已掌握”，从此阅读时将不会高亮显示它们，只留下真正的生词。

### 2. 阅读文章即读即查
*   在您的任意 Obsidian Markdown 文档中输入英文。
*   鼠标悬浮在任意高亮单词上，即可显示气泡释义。
*   **双击**单词：直接将单词标为“学习中”，系统会自动在后台抓取该词在当前文章中的完整句子，并在 `LangLearner/Cards/` 目录下为您生成一张带有双链的生词复习卡片。

### 3. 多媒体跟读与记笔记
*   在侧边栏切换至 **“Media”** (多媒体播放器) 或 **“Reader”** (RSS阅读/YouTube 字幕学习) 面板。
*   点击 **“📌 插入时间戳”** 按钮，编辑区会自动插入如 `[🎬 01:28](obsidian://lang-learner-media?url=...&t=88)` 格式的超链接。
*   在阅读笔记时，点击这个时间戳超链接，侧边栏视频会自动跳转到 `01:28` 播放。

### 4. 科学间隔复习
在侧边栏切换至复习面板，软件会根据 SM-2 算法为您筛选出今天需要复习的生词。点击“显示释义”翻转卡片，根据记忆效果进行 1~4 档评分，算法将智能安排下一次复习时间。

## ❓ 常见问题 (FAQ)

**Q：为什么点击单词发音没有声音？**
A：插件会优先使用有道/谷歌的在线真人语音。如果网络请求受限，系统会在后台自动无缝切换至您的电脑系统本地发音（TTS）路由。如果您使用的是 Linux 系统，请确保系统已安装 TTS 合成引擎；若是 macOS/Windows，请在侧边栏设置面板中选择合适的系统发音人。

**Q：如何迁移或备份我的生词本？**
A：所有的生词状态和学习数据全部存储在您 Obsidian 库根目录下的 `vocabulary.json` 中。您只需备份或同步这个 JSON 文件，您的生词库数据就永远不会丢失。

**Q：怎么更新插件？**
A：您可以在本地项目根目录中运行 `git pull` 拉取最新代码，重新运行 `npm run build`，然后将编译出来的三个文件替换掉插件目录下的旧文件即可。
