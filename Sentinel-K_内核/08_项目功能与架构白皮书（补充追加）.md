2. 新追加功能点详细清单 (V4.0 核心补强)
功能 12：本地轻量级声学强制对齐引擎 (Edge CAPT Engine)
WASM 硬件加速本地算分：无缝集成 40MB 级量化 Whisper-Tiny-EN 声学模型，利用 WebAssembly 在桌面端或移动端浏览器沙盒内实现纯前端、零资费、断网可用的语音特征提取。

音素级置信度检测：拒绝传统的粗暴声纹波形比对，通过动态时间规整（DTW）算法，将用户的实际发音与标准 GA（美音）/ RP（英音）靶点进行强制对齐（Forced Alignment），精准捕捉每一个音素的偏离概率。

内存无痕录音防爆：所有的纠音录音均在内存 ArrayBuffer 或临时 Blob URL 中高速级联，严禁向 Obsidian Vault 写入任何音频物理文件。当用户切换单词或关闭侧栏时，自动执行 URL.revokeObjectURL 彻底释放内存，达成零磁盘垃圾、零 I/O 同步冲突。

功能 13：云端认知算力降维级调用与 DeepSeek 肌肉诊断
Token 级成本极限压缩：本地评测结束后，摒弃高资费的云端语音大模型，仅将解构后的纯文本错误特征（如 “forward 中的卷舌音 /r/ 缺失”）作为 Payload。利用 DeepSeek API 极低的 Token 单价进行异步调度，将单次高频口语纠音的云端开销压低至 ￥0.005 元 以下。

中式发音肌肉习惯纠偏：大模型接收到本地物理评测指标后，不输出干瘪的分数，而是通过深度推理（Thinking Mode），输出地道发音肌肉纠偏指南（如提示用户舌头放置位置、是否错误地放平、如何利用“含水法”练习卷舌音等）。

双标准自适应发音矩阵：在 Vue 3 配置面板中开放 evaluationAccent 开关，支持 🇺🇸美式通用音 (GA) 与 🇬🇧英式标准音 (RP) 的一键切换。用户切换后，本地 G2P 靶点音标、系统 TTS 发音轨道以及发送给在线大模型的诊断基准同步强一致性联动。

A/B 键影子轨道实时回放：前端自动维护独占式 Web AudioContext 播放队列。用户可一键交替播放“标准母语人示范音”与“用户自身内存录音”，通过高保真听觉对比，加速建立口腔肌肉记忆。

3. V4.0 系统设计哲学追补
端云解耦与隐私最高防线 (Edge-Cloud Decoupling & Privacy Fortress):
用户的自主发音录音属于高度敏感的个人生物特征数据。V4.0 恪守“声音不出端、数据不上云”的红线，声音的拆解与判分全部在本地沙盒消化；上传到在线 DeepSeek 的只有脱敏后的纯文本错误编码。这不仅达成了省钱的极致，更为注重隐私的 Obsidian 核心骨灰级用户构筑了绝对安全的防线。

消费生词的主动催化伴写 (Output-Driven Activation):
将传统的“死背生词卡”升级为“主动消费生词”。当 EditorSuggest 捕获到用户在撰写英文工作流、日记或学术论文时，AI 推理层将联动内存影子词库，发现快要遗忘的生词，并在输入框下方动态诱导暗示（例如：“此处可将 reduce 替换为您生词本中处于复习临界点的 mitigate，记忆留存率可提升 30%”），通过高频的口语主动输出与伴写消费，彻底打通听、说、读、写闭环。
